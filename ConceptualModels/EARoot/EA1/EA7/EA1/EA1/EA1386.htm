The class TermRegisterEntry is a subclass of RE_RegisterItem (ISO 19135-1:2015, B.2.3.2). It specifies elements of information to be recorded for each terminological entry in a simple register, reference language subregister or submitted language subregister. <br/><br/>TermRegisterEntry is not used to record entries in a principal register. Entries in a principal register are recorded using instances of TermSubregisterDescription as described in ISO 19104:2016, 10.5.14.<br/>
