The class RE_AlternativeExpression is a data type used to provide information about a register item in an alternative language. Subclasses of RE_AlternativeExpression may be specified in order to add additional attributes appropriate for particular item classes.<br/><br/>EXAMPLE  A specification for a metadata element register could identify a subclass of RE_AlternativeExpression that would include the data dictionary fields specified in ISO 19115-1(obligation/condition, data type, and domain) as additional attributes.<br/>
