This package documents the schema components of GML that are not covered by the profile of the ISO 19100 series of International Standards defined in ISO 19136:2007 D.2. UML is used as the conceptual schema language to describe the additional elements in accordance with ISO/TS 19103. For details on the semantics of the additional classes see ISO 19136:2007 Clauses 7 to 19. The GML schema components have been grouped semantically based on the structure of these clauses and a package for each grouping is created. The additional types not documented in D.2 are defined in the corresponding package.<br/>
