This object represents a private property unit as an instance of the FullPartition_SpatialUnit. The whole Partition would include different object types, inserted in other layers, in such a way they together define the Full Partition contained within Continental territory of Portugal (in this example).<br/>In Portugal, the Partition would include, besides Cadastral Parcels:<br/><ul>
<li>Commons, which are referred as "Baldios" in Portugal;</li><li>Areas of collective shared use by local communities or the State, but not considered as "Baldios". Examples: ServingParcels, Public Domain, Municipal Domain;</li><li>Deferred Cadastre areas: where boundaries could not be surveyed or there are disputes;</li><li>Transitory Cadastre: where data is not harmonized with the Land Registry and the Land Tax Registry.</li></ul>
All those layers should obey to the ST_Contains topologic relationship.<br/>
