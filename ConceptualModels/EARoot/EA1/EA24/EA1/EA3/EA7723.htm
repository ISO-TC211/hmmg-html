An "LBS_SecurityCertificate" contains information that identifies the user whose data is being accessed, the user who is accessing that data (if different) and validation information that verifies that no unauthorized access is being made. The precise mechanism for this validation is determined by the security model being used by the service provider.  <br/><br/>
