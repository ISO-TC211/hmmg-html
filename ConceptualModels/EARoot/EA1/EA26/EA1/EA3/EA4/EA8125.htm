A temporal restriction limits the applicability ot the target to a particular instant or period of time. If an avoidance target expires, it need no longer be avoided. <br/>In a route request, a temporal restriction for a target specifices when that target should be visited. This may make some route request impossible, in which case, the service may return a best approximation, or simply a fail status.<br/>
