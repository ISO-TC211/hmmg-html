The LC_TopographicalAspects component is a subtype of LC_ClassCharacteristic. It has three optional attributes; <i>altitude</i>, <i>slope</i> and <i>slopeExposition</i>. The permitted values of the attribute <i>altitude</i> are described by the value object LC_PermittedRealValue. The permitted values of the attribute <i>slope</i> are described by the value object LC_PermittedPercentageValue. The permitted values of the attribute <i>slopeExposition</i> are described by the code list LC_SlopeExposition. This component may be used to refine the LC_LandCoverDescriptor metalanguage object corresponding to an entire LC_LandCoverClass with respect to topographical aspects to permit the generation of a more specific land cover feature class in a land cover classification system.<br/>
