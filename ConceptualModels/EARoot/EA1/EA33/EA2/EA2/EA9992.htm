The LC_GrowthForm metalanguage object is a subtype of LC_VegetationElement. It has two optional attributes <i>height</i> and <i>cover</i>. The permitted values of the attribute <i>height</i> are described by the values object LC_PermittedPositiveRealValue. The permitted values of the attribute <i>cover</i> are described by the values object LC_PermittedPercentageValue. This metalanguage object, when instantiated, allows the generation of a general land cover feature class in an LCCS for all growth form vegetation covered surfaces.<br/><br/>It has four subtypes LC_WoodyGrowthForms, LC_HerbaceousGrowthForms, LC_LichenAndMosses and LC_Algae.<br/>
