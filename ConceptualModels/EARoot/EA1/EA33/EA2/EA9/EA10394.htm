This example is a translation of a class from the Anderson (USGS) LCCS. The Anderson description is "evergreen forest land includes all forested areas in which the trees are predominantly those which remain green throughout the year. Both coniferous and broadleaved evergreens are included in this category". <br/><br/>The model shows that one layer is used composed by the LCML element Tree (with an extra characteristic Natural) is used. The LCML element Tree has two properties: Cover (from 30 to 100%) and Evergreen.<br/><br/>
