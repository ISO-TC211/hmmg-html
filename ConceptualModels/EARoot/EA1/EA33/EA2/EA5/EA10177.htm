The LC_WaterSupplyPeriod metalanguage object is a subtype of LC_CultivatedAndManagedVegetation. It has four optional attributes <i>rainfedPercentage</i>, <i>postfloodingPercentage</i>, <i>irrigationPercentage</i>, and <i>irrigationType</i>. The permitted values of the attributes <i>rainfedPercentage</i>, <i>postfloodingPercentage</i>, and <i>irrigationPercentage</i> are described by the values object LC_PermittedPercentageValue. The permitted values of the attribute type are described by the code list LC_IrrigationType. This component may be used to refine the metalanguage object corresponding to an LC_CultivatedAndManagedVegetation object to permit the generation of a more specific land cover feature class in an LCCS.<br/>
