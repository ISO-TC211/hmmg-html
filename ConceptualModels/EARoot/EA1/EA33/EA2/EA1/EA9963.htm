The code list LC_StratumPresenceType contains a list of types of stratum relationships. This list contains two types fixed and conditional. This list may be extended through registration. The attribute <i>presenceType</i> for the association class LC_InterStrataRelationship is optional and is by default the first value of this enumeration "<i>fixed</i>".The presence types are:<br/><br/>    <b><i>fixed</i></b><b> </b>(mandatory)- The LC_Stratum is always present in the LC_HorizontalPattern and indirectly in the LC_LandCover metaobject.<br/><br/>    <b><i>conditional</i></b><b> </b>(optional)- The existence of an LC_Stratum the LC_HorizontalPattern and indirectly in the LC_LandCover metaobject may exist dependant upon the existence of another LC_Stratum.<br/><br/>
