The component LC_WoodyLeafType is related to LC_WoodyGrowthForm by the relation <i>leafType</i>. It has three subtypes LC_NeedleLeaf, LC_Aphillious and LC_Broadleaf. It has one optional attribute <i>percentage</i>. The permitted values of the attribute <i>percentage</i> are described by the values object LC_PermittedPercentageValue with the constraint that the {sum of all percentage attributes for every LC_WoodyGrowthLeafType link for a single LC_WoodyGrowthForm must not be more than 100%}. The component LC_WoodyLeafType may be used to refine the metalanguage object LC_WoodyGrowthForm to permit the generation of a more specific land cover feature class in an LCCS. <br/>
