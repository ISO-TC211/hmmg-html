The initial (root) class of the LCML is the LC_LandCoverClassificationSystemMetaLanguage object. This object is composed of all the land cover elements that when serialzed describe the classes that make up a classification system as given by LC_LandCoverClassificationSystem. <br/><br/>The LC_LandCoverClassificationSystemMetaLanguage object has one relationship meta<i>SystemCollection</i>. LC_LandCoverClassificationSystemMetaLanguage is an aggregation of the land cover objects LC_LandCover.<br/>
