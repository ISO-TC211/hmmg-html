The LC_LandCoverClass metalanguage object is the result of the serialization of the LC_LandCover object; that is, it is related to LC_LandCoverDescriptor by a dependency relationship that indicates that the LC_LandCoverClass is described by the seralization of the metalanguage objects that compose the LC_LabdCoverDescriptor. It corresponds to a single land cover class in a Land Cover Classification System. <br/>This object has a relationship <i>systemCollection</i> with the object LC_LandCoverClassificationSystem.<br/>This object is a subtype of the class CL_LegendClass as defined in part 1 of this International Standard; that is, it is a specialization of a general classification system Legend Class for the use of land cover.<br/><br/>
