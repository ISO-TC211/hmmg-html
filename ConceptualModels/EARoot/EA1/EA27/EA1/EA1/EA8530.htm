The class RegisterStakeholder specifies information about the register stakeholder, which is either a manager, owner or submitter. A stakeholder can serve one or more roles.<br/><br/>NOTE This International Standard does not require that a register stakeholder name be unique, since an stakeholder will, in general, have had a name before undertaking a role in connection with a register.<br/>
