Curves are the primitive 1D geometry in a map. Each pair of curves in a single map do not intersect one another except at endpoints, which are point instances in the same map. Internal positions may be represented by points in other maps, but may not be in the same map. NL_Curve is the basic curve type for the package, consisting of some number of curve segments (NL_Segment) containing coordinate descriptions. <br/>
