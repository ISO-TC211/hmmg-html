This class inherits three attributes: name, technicalStandard and alternativeLanguages. <br/>For the LCCS register described in this part of ISO 19144 the attribute "name" shall contains the text string " LC_ElementDescriptorClass " to identify the registered class.  <br/>The attribute "technicalStandard" shall contain the following:<br/><br/><ul>
<li><i>title</i>: “ISO 19144-2, Geographic information - Classification Systems - Part 2: Land Cover Meta Language (LCML)”</li><li><i>alternateTitle</i>: “ISO 19144-2”</li><li><i>date</i>: xxxx<sup>1</sup></li><li><i>otherCitationDetails</i>: “ISO 19144-2 clause 9.3”</li></ul>
The attribute "alternativeLanguage" is optional.<br/><br/><sup>1</sup> To be determined when the standard is published.<br/><br/>
