This class is used for routes defined by a geometry (a composite curve) and start and stop positions on that geometry. A route is calculated from and optimizes a cost function from the set of possible routes that traverse a set of target positions. The recalculate function determines a new route that optimizes the new cost function from the passed target position to the end of the route. If the cost function has not changed the new route should coincide with old one.<br/><br/>The two canonical representations of a route are 1) an ordered list of directed links to be traversed<br/>and 2) an ordered list of maneuver to be made. Each maneuver advisory or route instruction is associated to either a link, a maneuver or a turn of the route.<br/>
