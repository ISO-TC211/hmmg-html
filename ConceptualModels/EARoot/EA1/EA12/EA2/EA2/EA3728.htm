Details:
Association from a general derived coordinate reference system that uses this single coordinate reference system as its source. This association is used when one CRS can only exist when a base CRS and a defining conversion that relates the derived CRS to its base CRS both exist.  For instance, a projected CRS cannot exist without a base geodetic CRS and a map projection (coordinate conversion). <br/>
