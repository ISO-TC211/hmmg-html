derived coordinate reference system which has a geodetic (usually geographic) coordinate reference system as its base CRS, thereby inheriting a geodetic reference frame, is converted using a map projection, and has a Cartesian coordinate system, usually two-dimensional but may be three-dimensional<br/>Note: In the 3D case the base geographic CRSs ellipsoidal height is passed through unchanged and forms the vertical axis of the projected CRS's Cartesian coordinate system.<br/>

Association from a projected coordinate reference system to its base geodetic coordinate reference system. 		Name: baseCRS
geodetic or geographic coordinate reference system that is the baseCRS for this projected coordinate reference system
Details:
Association from a projected coordinate reference system to its base geodetic coordinate reference systems. This association is used when one CRS can only exist when a base CRS and a defining coordinate conversion that relates the projected CRS to the base CRS both exist.  A projected CRS cannot exist without a base geodetic CRS and a map projection (coordinate conversion). <br/>
