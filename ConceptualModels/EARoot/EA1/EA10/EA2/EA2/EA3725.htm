Details:
Association from a projected coordinate reference system to its base geodetic coordinate reference systems. This association is used when one CRS can only exist when a base CRS and a defining coordinate conversion that relates the projected CRS to the base CRS both exist.  A projected CRS cannot exist without a base geodetic CRS and a map projection (coordinate conversion). <br/>
