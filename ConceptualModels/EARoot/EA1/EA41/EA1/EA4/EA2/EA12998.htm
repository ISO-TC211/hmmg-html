Side effects are conditions that cause a logically unrelated action to occur. Such actions are usually required by the contract that the licence represent. The evaluate function, inherited from Condition, will return true if the side effect does not uncover any reason to interrupt the flow. If a side effect has a time set to after the act, i.e. if the effect is to occur on successful completion of the licensed act., the return value of the Condition::evaluate function should always be true. <br/>
