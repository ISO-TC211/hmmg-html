SY_LineStyle specializes SY_Presentation as an abstract line style.  The line style has a coordinate reference system defined at every point along its path, which has an x-axis that is tangential to the path at that point. The line style also has a one dimensional coordinate reference system that is define along the path of the line style.<br/>
