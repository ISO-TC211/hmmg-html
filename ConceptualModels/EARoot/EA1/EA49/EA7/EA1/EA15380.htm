The NL_Map is a maximal 2 dimensional composite surface representing some portion of the earth's surface. The features in a Map are represented by subcomplexes (usually composites) of the map that represent the point, curve or area of a feature.<br/>The map represents the limits of the topological structure of the data represented. If the maps that these objects represent are contained in a more complete larger map, then some sort of multiple representation must be used to  segregate the topological structures from one another.<br/>
